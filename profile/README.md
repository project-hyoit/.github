# 효잇 - 가족을 연결하는 AI 기반 소통 앱

효잇은 부모와 자녀를 연결해주는 모바일 애플리케이션입니다.

단순 메시징을 넘어, 챗봇과 게임을 통해 자연스럽게 대화가 이어지도록 설계했습니다.

---

## 왜 만들었는가

가족 간 대화는 점점 줄어들고 있지만,

메신저는 오히려 더 형식적인 소통 도구가 되는 경우가 많습니다.

효잇은

- “억지로 하는 대화”가 아니라
- “자연스럽게 이어지는 대화”를 만드는 것에 집중했습니다.

---

## 핵심 기능

### 1. 가족 연결

- 6자리 코드 기반 연결
- 가족 구성원 관리 (조회 / 삭제)

가족이라는 관계를 앱 안에서 하나의 네트워크로 묶는 구조입니다.

---

### 2. 홈

- 날씨 정보 제공 (기상청 API)
- 주요 기능 진입
- 가족 상태 확인

앱을 켰을 때 가장 먼저 “가족 상황”을 볼 수 있도록 구성했습니다.

---

### 3. 게임

단순한 미니게임이 아니라

> 대화를 유도하는 장치로 설계했습니다.
> 
- 숫자 뒤집기 게임
- 난이도 / 목숨 설정 가능
- 플레이 기록 저장

---

### 4. 챗봇

OpenAI 기반 챗봇을 사용했습니다.

- 텍스트 채팅
- 음성 입력 (STT)

단순 질문 응답이 아니라,

가족 간 대화의 시작점을 만들어주는 역할을 합니다.

---

### 5. 프로필

- 프로필 관리
- 가족 상태 표시
- 접속 상태 시각화

→ 온라인 여부를 직관적으로 확인 가능

---

## 인증

- 카카오 OAuth 로그인
- 사용자 정보 등록

로그인 허들을 낮추는 데 집중했습니다.
