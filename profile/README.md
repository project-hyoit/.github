# 효잇 - 👨‍👩‍👧‍👦 가족을 연결하는 AI 기반 소통 플랫폼 👨‍👩‍👧‍👦

효잇은 **가족 간 소통을 돕는 모바일 애플리케이션 서비스** 입니다.

부모와 자녀가 앱을 통해 연결되어 메시지, 챗봇, 간단한 게임 기능을 이용할 수 있으며 가족 간의 디지털 커뮤니케이션을 지원합니다.

---

## 📌 프로젝트 목적

효잇은 다음과 같은 목표를 가지고 개발되었습니다.

- 가족 간 **간편한 연결 시스템 제공**
- 앱 기반 **소통 플랫폼 구축**
- 챗봇과 게임을 통한 **사용자 경험 향상**

사용자는 계정을 생성하고 가족을 연결한 뒤 다양한 기능을 사용할 수 있습니다.

---

## 🧩 주요 기능

로그인 및 계정 생성

- 카카오 OAuth 로그인
- 사용자 정보 등록
- 가족 연결 코드 생성

카카오 계정을 이용하여 간편하게 로그인할 수 있습니다.

---

## 1️⃣ 가족 연결 시스템

- 6자리 초대 코드 생성
- 가족 연결
- 연결된 가족 정보 조회
- 가족 삭제

가족 구성원을 앱에서 연결하여 서로의 정보를 확인할 수 있습니다.

---

## 2️⃣ 홈 화면

홈 화면에서는 다음 정보를 확인할 수 있습니다.

- 오늘의 날씨
- 주요 기능 이동 버튼
- 연결된 가족 상태 확인

날씨 정보는 **기상청 Open API**를 사용하여 제공합니다.

---

## 3️⃣ 게임 기능

앱 내에서 간단한 게임을 즐길 수 있습니다.

### 숫자 뒤집기 게임

- 난이도
    - 쉬움, 보통, 어려움
- 목숨
    -  8개, 6개, 4개
- 카드 개수
    - 16개

게임 기록은 서버로 전송되어 저장됩니다.

---

## 4️⃣ 챗봇 기능

OpenAI 기반 챗봇을 제공합니다.

기능

- 텍스트 채팅
- 음성 입력 (STT)

음성 버튼을 통해 음성을 텍스트로 변환하여 메시지를 보낼 수 있습니다.

---

## 5️⃣ 프로필 관리

사용자는 자신의 프로필을 확인하고 관리할 수 있습니다.

- 프로필 조회
- 프로필 사진 수정
- 연결된 가족 확인
- 접속 상태 표시
- 로그아웃

접속 상태는 색상으로 표시됩니다.

- 접속 중 → 녹색
- 비접속 → 회색
